CONCEPTEUR DESIGNER UI - BC01 (RNCP 35634)

Contenu: 57 heures

éligible CPF

BC01 - Concevoir les éléments graphiques d'une interface et de supports de communication. Cette formation vous enseignera les principes fondamentaux du design d'interface, tels que la composition visuelle, la typographie, la couleur, la hiérarchie visuelle et la mise en page, tout en mettant l'accent sur la conception centrée sur l'utilisateur. Vous utiliserez des outils tels que Sketch, Adobe XD ou Figma pour créer des maquettes, des prototypes interactifs et collaborer efficacement avec les développeurs et les autres membres de l'équipe de conception. En tant que concepteur designer UI, vous jouerez un rôle essentiel dans la création d'expériences numériques engageantes, en travaillant en étroite collaboration avec les développeurs, les UX designers et les autres parties prenantes pour transformer les besoins des utilisateurs en interfaces visuelles fonctionnelles et esthétiques.

Contenu 75 heures

éligible CPF

JAVASCRIPT

Contenu 47 heures

BC01 - Elaborer une proposition graphique. La formation en JavaScript vous permet de créer des fonctionnalités interactives sur les sites web. Ce langage polyvalent est utilisé pour gérer les éléments de la page, valider les formulaires, communiquer avec des API, etc. En suivant cette formation, vous apprendrez les bases de la programmation JavaScript, tels que les variables, les fonctions, les boucles et les conditions. Vous découvrirez également les fonctionnalités avancées, comme la manipulation du DOM, les requêtes AJAX et l'utilisation de frameworks populaires tels que React, Angular et Vue.js. Le JavaScript est largement utilisé dans l'industrie du développement web, offrant de nombreuses opportunités d'emploi. Les développeurs JavaScript sont recherchés pour créer des applications web interactives, des jeux en ligne, des applications mobiles hybrides, etc.
